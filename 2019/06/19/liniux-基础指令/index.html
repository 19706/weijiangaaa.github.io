[linux]基础指令

2、Linux基本操作

以下都是从接触Jetson Nano开始，学习Linux的笔记：

2.1简介

Jetson Nano官方提高的系统镜像中预置了基于Ubuntu16.04的LTS系统，其中包含了吧啦吧啦一堆（）
这里是一些常见的问题说明，新手需要注意：

  • Linux下的文件，没有拓展名的概念，所以看到txt不要想当然的就认为是一个文本文件；
  • 同时，文件名是区分大小写的；

    2.2基础常用指令

    2.2.1 文件管理指令

    cd ~ #切换至home目录
    pwd #查看当前的工作目录
    cd / #切换至root目录
    cd .. #回到上一次目录

ls
ls f* #查看以f开头的所有文件和子目录
ls -a #查看包含隐藏文件的文件和子目录

cp 1.txt 2.txt #将文本1复制成文本2

mv

rm

2.2.2 编辑文本文件

通过nano编辑器来编辑文本文件

sudo nano 1.txt

查看文本文件内容

cat 1.txt
more 1.txt

将命令行文件导至文本文件中

echo “hello world” > 1.txt

2.2.2 目录管理

创建目录

mkdir

删除目录

rm

2.2.3 文件权限

查看文件权限

  • -rwx rwx rwx
    代表：文件；所有者可读、可写、可运行；群组。。。；其他用户。。。；

    ls -l

修改文件权限

chmod 777 1.txt

其中的(421)(421)(421)->777

变更文件所有者与群组
文本1的所有者变更为tina

chown tina 1.txt
文本1的所有者变更为root，但文件群组变更为tina
chown root:tina 1.txt

2.2.4 系统管理

超级用户权利来运行工作

sudo

修改密码

passwd

2.2.5 升级操作系统

sudo apt-get update
sudo apt-get upgrade

重启系统，确保系统使用最新的升级套件

reboot

2.2.6 安装软件套件

sudo apt-get install

2.2.7 自动运行程序

修改【/etc/rc.local】文件，在文件内容的最后加入以下信息：

/usr/bin/python /home/pi/button.py &

其中【&】表明以后台方式运行程序

2.2.7 Shell Script

Linux标准的Shell为BASH；
Script语言的特色是，编写成文本文件后，不需要事先编译，而是运行时直接解译每一行的程序内容。

nano sh01

输入想依次执行的shell命令

#!/bin/bash

chmod 777 sh01

./sh01
