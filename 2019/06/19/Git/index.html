Git

Git是一个开源的分布式版本控制系统，可以有效、高速地处理从很小到非常大的项目版本管理。

个人目前的常用途：
1、目前绝大多数优秀项目的源码都会开源在github上，为了clone；
2、个人博客是搭载github上的，git是基础命令工具，同是为的是素材的网络地址；
3、为了项目源代码的开源共享。

  • 比喻：【reset】改变了历史，包括其中的人物命运也随历史的不同而不同；【checkou】不改变历史，而是可以单独可查个人在不同版本历史的命运。就好像是针对历史中某个人物的命运改变，即只回退了单一文件的版本，
