九、卷积神经网络

（Convolutional Neural Networks, CNN）

在图片和语言方面有更优的测试效果

计算器图像识别、视频分析、NLP、AlphaGo等等

运作分析：
图片分析
图片的矩阵
优势在于，卷积在于区域处理，增加连续性，增加理解，批量过滤器

pixels->edges->object parts->objects model

原始图像经过过滤器之后，产生一个高度更高，但是长和宽更小的新图像，该图像包含了一些边缘信息。

然后将最后压缩的图像，经过普通的全连接层，经过分类，便可以对图像进行分类。

池化：在压缩时候，会丢失信息，所以池化出现，在卷积时候不压缩长和宽，保留更多信息，压缩的工作交给池化，附加的工作提高了准确性。

常用的结构：image-convolution-max pooling-convolution-max pooling-fully-fully-classifier

MINST手写数据识别

导入常规的模块；

torchvision包含了图片的数据库

EPOCH说明只训练一次，节省时间；
BATCH_SIZE说明没一小批次的数据有50个；
LR说明学习率是0.001；
DOWNLOAD_MNIST = Flase说明已经下载过数据了，所以赋值为False；

root是指下载的路径；

train的有6w多个，test的数据有1w多个；

transform要将下载的数据，可能是pixels，或者是numpy()的，我们要改为Tensor，附加到train_data中去；一般图像的范围是【0~255】，这里将数据的范围归一化到【0~1】之间。

可视化的例子；

[:2000]取2000个样本，并/255将数据归一化到[0,1]；

1、nn.Conv2d是一个卷积的过滤器，有长宽高，高度H是指有多少个filter，说明卷积一下能提取出H个特征值

  • in_channels=1这个图片有多少个层，这里输入的图像是灰度图，只有1层；
  • out_channels=161次扫描共有16个filter，一次扫描能提取出16个特征；
  • kernel_size=5过滤器的尺寸，高和宽都是5个像素点；
  • stride=1过滤器一次扫描移动多少步；
  • padding=2图像边缘补充多少个像素点，围一圈，padding=(kernel_size-1)/2，这个公式说明卷积之后的图像大小和之前的大小一致；

2、nn.ReLU

3、nn.MaxPool2d

  • 此时的图片已经很厚了，通过MaxPool2d提取出有用的信息，变成更小但是高度不变的图片；
  • nn.Linear的作用就是全连接层，3维的数据要进行展平

x = self.conv1(x)
x = self.conv2(x) # (batch_size, 32, 7, 7)

  • 前向传播的过程是，把训练数据x输入到搭建好的conv1和conv2中去，在这种卷积和池化的方式之下，4维的数据(batch_size, 32, 7, 7)在这些层之间完成了前向传播；

x = x.view(x.size(0), -1) # flatten the output of conv2 to (batch_size, 32 7 7)

  • 在输入liner层之前，我们要做数据的变换，将(batch_size, 32, 7, 7)变为(batch_size, 32 * 7 * 7)，这样才符合输入liner层的数据格式，因为其是全连接层；
  • x.size(0)即保持batch_size，-1代表的是把后面的都压缩在一起。
