[linux]基础指令

2、Linux基本操作

以下都是从接触Jetson Nano开始，学习Linux的笔记：

2.1简介

Jetson Nano官方提高的系统镜像中预置了基于Ubuntu16.04的LTS系统，其中包含了吧啦吧啦一堆（）
这里是一些常见的问题说明，新手需要注意：

  • Linux下的文件，没有拓展名的概念，所以看到txt不要想当然的就认为是一个文本文件；
  • 同时，文件名是区分大小写的；

    2.2基础常用指令

    2.2.1 文件管理指令

    cd ~ #切换至home目录
    pwd #查看当前的工作目录
    cd / #切换至root目录
    cd .. #回到上一次目录

ls
ls f* #查看以f开头的所有文件和子目录
ls -a #查看包含隐藏文件的文件和子目录

cp 1.txt 2.txt #将文本1复制成文本2

mv

rm

2.2.2 编辑文本文件

通过nano编辑器来编辑文本文件

sudo nano 1.txt

查看文本文件内容

cat 1.txt
more 1.txt

将命令行文件导至文本文件中

echo “hello world” > 1.txt

2.2.2 目录管理

创建目录

mkdir

删除目录

rm

2.2.3 文件权限

查看文件权限

  • -rwx rwx rwx
    代表：文件；所有者可读、可写、可运行；群组。。。；其他用户。。。；

    ls -l

修改文件权限

chmod 777 1.txt

其中的(421)(421)(421)->777

变更文件所有者与群组
文本1的所有者变更为tina

chown tina 1.txt
文本1的所有者变更为root，但文件群组变更为tina
chown root:tina 1.txt

2.2.4 系统管理

超级用户权利来运行工作

sudo

修改密码

passwd

2.2.5 升级操作系统

sudo apt-get update
sudo apt-get upgrade

重启系统，确保系统使用最新的升级套件

reboot

2.2.6 安装软件套件

sudo apt-get install

2.2.7 自动运行程序

修改【/etc/rc.local】文件，在文件内容的最后加入以下信息：

/usr/bin/python /home/pi/button.py &

其中【&】表明以后台方式运行程序

2.2.7 Shell Script

Linux标准的Shell为BASH；
Script语言的特色是，编写成文本文件后，不需要事先编译，而是运行时直接解译每一行的程序内容。

nano sh01

输入想依次执行的shell命令

#!/bin/bash

chmod 777 sh01

./sh01

[人脸识别]一、Haar Cascade

1、Haar Cascade

时间：2001年。

说明：最经典的算法，被Matlab和OpenCV引用作为库。

概况：基于Haar特征的cascade分类器。它是一种机器学习方法，通过许多正负样例中训练得到cascade方程，然后将其应用于其他图片。

前言：

目标检测的基本原理（yolo之前的那些方案）：

1、先通过训练集学习一个分类器；

2、然后在测试图像中以不同scale的窗口滑动扫描整个图像；

3、每次扫描做一下分类，判断一下当前的这个窗口是否为要检测的目标。

  • 算法的核心是分类，分类的核心一个是用什么特征，一个是用哪种分类器。
  • Haar特征/矩形特征，计算Haar特征个数，是使用黑色的所有像素值和减去白色的所有像素值和，24x24尺寸的滑动窗口特征数就超过160000。

  • Integral Image，来重新表示一幅图像，把计算量一下就缩减了一个量级。

  • AdaBoost分类器，是把多个弱分类器结合起来形成一个强分类器，是这些弱分类器的加权和。每个弱分类器的准确率并不高，但最后形成的强分类器的准确率就很高了，有很好的泛化能力，从超过16万个的特征中选择最好的特征，只需要200个特征的分类器在检测中的精确度达到了95%。

    ”因此，我们将每个特征都应用于所有训练集图片中，对于每个特征，找出人脸图片分类效果最好的阈值。显然，分类会有误分类，我们选择分类错误率最小的那些特征，也就是说这些特征可以最好的将人脸图片和非人脸图片区分开。(实际上没有那么简单，每个图片初始时给定相同的权重。每次分类后，提升被分类错误图片的权重。然后再根据新的权重分类，计算出新的错误率和新的权重，直到错误率或迭代次数达到要求即停止。“

  • Cascade 分类器，有人后续引入，提出级联分类器的方法，窗口在第一个阶段就检测失败了，那么就直接舍弃它，无需考虑剩下的特征。 级联分类器的关键是强分类器的次序安排：把简单的、False Negative低的安排在最前面。 这样可以快速判断出那些不相关的区域，把更多的计算量用在潜在区域.
    • False Positive：把合法的判断成非法的，译为“误报”。（相当于医学上的假阳性）
    • False Negative：把非法的判断成合法，译为“漏报”。（相当于医学上的假阴性）

《Rapid Object Detection using a Boosted Cascade of Simple Features》
原文链接：https://ieeexplore.ieee.org/document/990517
参考链接：https://blog.csdn.net/u012116229/article/details/45057937

[c++]c++笔记记录

存储变量类型：

1、register 存储类用于定义存储在寄存器中而不是 RAM 中的局部变量，且不能对它应用一元的 ‘&’ 运算符。

2、static 修饰局部变量可以在函数调用之间保持局部变量的值。
static 修饰符也可以应用于全局变量。当 static 修饰全局变量时，会使变量的作用域限制在声明它的文件内。

3、extern 是用来在另一个文件中声明一个全局变量或函数。
无法初始化的变量。
全局变量对所有的程序文件都是可见的。

函数参数：

调用类型 描述
传值调用 修改函数内的形式参数对实际参数没有影响。
指针调用 该方法把参数的地址复制给形式参数，修改形式参数会影响实际参数。
引用调用 该方法把参数的引用复制给形式参数，修改形式参数会影响实际参数。

字符串：

指针：

动态内存分配。
指针是一个变量，其值为另一个变量的地址。
所有指针的值的实际数据类型，不管是整型、浮点型、字符型，还是其他的数据类型，都是一样的，都是一个代表内存地址的长的十六进制数。
定义一个指针变量->把变量地址赋值给指针->访问指针变量中可用地址的值。
